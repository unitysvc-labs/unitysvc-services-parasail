# Parasail

[Parasail](https://parasail.io/) provides affordable, high-performance serverless inference for open-weight models including DeepSeek, Qwen, Llama, and more.

## Usage

Access models via the OpenAI-compatible API using your Parasail API key. All serverless models support chat completions, with pricing based on parameter size per million tokens.
